ViTaSCOPE is a neural implicit representation that is trained entirely in simulation to integrate vision and touch for in-hand object pose and extrinsic contact estimation.

ViTaSCOPE

How do we leverage complementary visuo-tactile feedback to infer extrinsic contact between grasped object and the environment?

Object module (O): A signed distance field (SDF) that represents the 3D object geometry.

Tactile module (T): A shear vector field that characterizes the extrinsic contact configuration.

Contact module (C): A contact field that represents the contact probability across the object surface.
